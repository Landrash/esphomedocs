Hi there! This is the documentation for esphomeyaml, a project that aims to make using ESP8266/ESP32 boards with Home Assistant very easy with no programming experience required.

To get started, please see the Getting Started Guide. Also be sure to check out Frequently Asked Questions.

You can find the source for this project on Github, and the core of this project, esphomelib is also hosted on Github.
